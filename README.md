## Minimum Requirement

-   PHP 5.6
-   WordPress 4.4+

## Installation

-   Clone the repository inside `/wp-content/plugins/`
-   CD into folder `cd wp-erp` and run `composer install` and then `composer dump-autoload -o`
-   Lastly run `npm install` and then `npm run build`
-   Activate the plugin
